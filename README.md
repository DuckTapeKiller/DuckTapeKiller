◈  **Philosophy**
I build for the user who treats their vault as a workspace, not a mood board. Every plugin and theme I ship follows the same operating principle: the interface should disappear. Controls should be where you expect them. Typography should not demand attention. Structure should be visible without being decorative.

Sharp corners. High contrast. No soft gradients pretending to be philosophy.

▣  **Themes**
Two distinct takes on the same principle: Obsidian as a structural tool.

■  **Brutalist**  
![Obsidian](https://img.shields.io/badge/Obsidian-Theme-black?style=flat-square&logo=obsidian&logoColor=white) 
A stark, geometric theme for writers and heavy readers. Built for users who live in their vault. Gentle on the eyes over long sessions, with a readable maximum content width of 800px and a curated embedded typeface stack: Marcellus and IBM Plex Serif for body, Space Mono for code and UI.
[DuckTapeKiller/Brutalist](https://github.com/DuckTapeKiller/Brutalist)

■  **Aubade**  
![Obsidian](https://img.shields.io/badge/Obsidian-Theme-black?style=flat-square&logo=obsidian&logoColor=white)
A retro-brutalist theme. Zero tolerance for roundness. Where Brutalist is refined, Aubade is structural. A nuclear anti-roundness policy means no plugin or native element is permitted to introduce a rounded corner. Borders are 2px thick throughout.
[DuckTapeKiller/obsidian-aubade](https://github.com/DuckTapeKiller/obsidian-aubade)

▣  **Plugins**
Six plugins, all independently maintained. Each solves one problem precisely.

■  **Easy View**  
![Obsidian](https://img.shields.io/badge/Obsidian-Plugin-black?style=flat-square&logo=obsidian&logoColor=white)
One-click view controls from the status bar. Puts font size, color, light/dark toggle, and focus mode buttons directly in the Obsidian status bar.
[DuckTapeKiller/obsidian-easyview](https://github.com/DuckTapeKiller/obsidian-easyview)

■  **Reader Highlighter + Tags**  
![Obsidian](https://img.shields.io/badge/Obsidian-Plugin-black?style=flat-square&logo=obsidian&logoColor=white)
A Medium-grade annotation layer built directly into Reading View. Floating toolbar for markup without switching to Edit mode.
[DuckTapeKiller/obsidian-reader-highlighter-tags](https://github.com/DuckTapeKiller/obsidian-reader-highlighter-tags)

■  **Book Search Plus**  
![Obsidian](https://img.shields.io/badge/Obsidian-Plugin-black?style=flat-square&logo=obsidian&logoColor=white)
Multi-provider book metadata importer (Goodreads, OpenLibrary, Google Books, Calibre).
[DuckTapeKiller/obsidian-book-search-plus](https://github.com/DuckTapeKiller/obsidian-book-search-plus)

■  **Bulk Tag Manager**  
![Obsidian](https://img.shields.io/badge/Obsidian-Plugin-black?style=flat-square&logo=obsidian&logoColor=white)
Vault-wide tag standardization dashboard. Rename, merge, and bulk convert casing across all files.
[DuckTapeKiller/obsidian-bulk-tag-manager](https://github.com/DuckTapeKiller/obsidian-bulk-tag-manager)

■  **Kindle Importer Plus**  
![Obsidian](https://img.shields.io/badge/Obsidian-Plugin-black?style=flat-square&logo=obsidian&logoColor=white)
Intelligent Kindle highlight appending to existing book notes.
[DuckTapeKiller/obsidian-kindle-importer-plus](https://github.com/DuckTapeKiller/obsidian-kindle-importer-plus)

■  **Jellyfin Integration**  
![Obsidian](https://img.shields.io/badge/Obsidian-Plugin-black?style=flat-square&logo=obsidian&logoColor=white)
Browse and import Jellyfin media server metadata into a visual poster grid within Obsidian.
[DuckTapeKiller/obsidian-jellyfin-integration](https://github.com/DuckTapeKiller/obsidian-jellyfin-integration)
